# Lab 10-12: Introduction to 65C816 CPU

## Lab Overview

In this lab, students will study the 65C816 CPU architecture, compare it to SAP-1, and complete mapping exercises to understand the transition to commercial microprocessors.

## Learning Objectives

- Study 65C816 architecture and features
- Compare SAP-1 to 65C816
- Understand 16-bit and 24-bit concepts
- Complete component mapping exercises
- Understand addressing modes

## Prerequisites

- Completed Term 1 (Enhanced SAP-1)
- Understanding of CPU architecture
- Understanding of memory systems
- Understanding of instruction execution

## Materials

### Components
- 65C816 datasheet (WDC)
- SAP-1 documentation
- Comparison worksheets
- Reference materials

### Tools
- Documentation software
- Research materials
- Comparison templates

## Safety

- Study and research activity
- No hardware construction yet

## Lab Sessions

### Session 1: 65C816 Architecture Study (Week 13)

**Objective**: Study 65C816 architecture

**Steps**:
1. Read 65C816 datasheet
2. Identify key features
3. Study register set
4. Study addressing modes
5. Study instruction set
6. Document findings
7. Create architecture summary

**Expected Result**: Understanding of 65C816 architecture

**Troubleshooting**:
- If datasheet unclear: Use additional resources, ask questions
- If concepts difficult: Review SAP-1 concepts, build understanding
- If questions: Research, discuss with class

### Session 2: SAP-1 to 65C816 Comparison (Week 14)

**Objective**: Compare SAP-1 and 65C816

**Steps**:
1. Create comparison table
2. Map SAP-1 components to 65C816
3. Compare instruction sets
4. Compare addressing modes
5. Compare capabilities
6. Identify similarities
7. Identify differences
8. Document comparison

**Expected Result**: Complete comparison understanding

**Troubleshooting**:
- If mapping unclear: Review both architectures
- If differences confusing: Focus on similarities first
- If comparison incomplete: Add more categories

### Session 3: Component Mapping Exercise (Week 15)

**Objective**: Map SAP-1 components to 65C816

**Steps**:
1. Map Program Counter
2. Map Instruction Register
3. Map Registers (A, B â†’ A, X, Y)
4. Map Flags
5. Map ALU
6. Map Memory
7. Map Control Unit
8. Document mappings

**Expected Result**: Complete component mapping

**Troubleshooting**:
- If mapping unclear: Review component functions
- If no equivalent: Note differences
- If mapping wrong: Review and correct

### Session 4: Addressing Modes Study (Week 16)

**Objective**: Understand 65C816 addressing modes

**Steps**:
1. Study immediate mode
2. Study direct mode
3. Study indexed mode
4. Study indirect mode
5. Compare to SAP-1 addressing
6. Understand benefits
7. Document addressing modes

**Expected Result**: Understanding of addressing modes

**Troubleshooting**:
- If modes unclear: Study examples
- If benefits unclear: Compare to SAP-1
- If too complex: Focus on basic modes first

## Assessment
