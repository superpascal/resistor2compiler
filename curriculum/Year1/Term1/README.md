# Year 1, Term 1: Electronics Basics and Lab Skills

## Overview

Term 1 establishes the foundation for all future work. Students learn to work safely in the lab, understand basic electronics, and build their first circuits.

## Learning Outcomes

By the end of Term 1, students will:
- Work safely in electronics lab
- Use breadboards, multimeters, and basic tools
- Understand voltage, current, and resistance
- Build timing circuits with 555 timers
- Understand and build Boolean logic gates
- Build combinational logic circuits (adders, multiplexers)

## Week-by-Week Structure

- **Week 1**: Documentation Standards, Lab Introduction & Safety
- **Weeks 1-2**: Electricity Basics, Power Calculations, LED Circuits, Ohm's Law
- **Week 2**: Systematic Troubleshooting
- **Weeks 2-3**: Soldering Skills
- **Weeks 3-4**: Breadboard Best Practices, Resistors
- **Weeks 4-5**: Component Datasheets
- **Weeks 5-6**: Diodes Comprehensive
- **Week 6**: Oscilloscope Usage
- **Weeks 6-8**: Boolean Logic and Logic Gates
- **Weeks 7-8**: Advanced Capacitors, Inductors, IC Fundamentals, Switches/Buttons, Voltage Regulators
- **Weeks 8-9**: Fuses, Connectors/Headers, Wire and Cable
- **Weeks 9-12**: Combinational Logic Circuits (Adders, Multiplexers, Decoders)
