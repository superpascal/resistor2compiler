# Year 2, Term 2: Advanced Control Systems and I/O

## Overview

Term 2 of Year 2 extends control concepts and introduces input/output systems. Students build more complex state machines, explore microcode concepts, and learn to interface with external devices.

## Learning Outcomes

By the end of Term 2, students will:
- Build more complex state machines and sequencers
- Understand microcode ROM concepts
- Implement address decoding for multiple devices
- Interface with simple I/O devices (switches, LEDs, displays)
- Understand memory-mapped I/O
- Build more sophisticated control systems

- **Formative**: Weekly practical work
- **Summative**: Advanced state machine project (Weeks 10-12)
- **Summative**: Address decoding project (Weeks 13-15)
- **Summative**: I/O system project (Weeks 16-18)
- **Final**: Complete advanced system demonstration

## Prerequisites

Students must have completed Year 2 Term 1 and understand:
- Basic state machines and sequencers
- Memory concepts and MAR
- Control signals and coordination
- Bus protocols

## Key Concepts

- **Advanced State Machines**: More complex sequencing
- **Microcode**: Storing control sequences in ROM
- **Address Decoding**: Selecting multiple devices
- **Memory-Mapped I/O**: Treating I/O as memory
- **I/O Interfaces**: Connecting external devices

## Milestones

- **Week 12**: Microcode ROM working
- **Week 15**: Address decoding implemented
- **Week 18**: I/O system complete and tested
