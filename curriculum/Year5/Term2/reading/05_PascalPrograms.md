# Chapter 5: Pascal Program Development

## Introduction

Pascal programs demonstrate the complete system capabilities. This chapter explains how to develop programs for the 65C816 system.

## Pascal Programming on Hardware

**Programming approach**:
- Write Pascal programs
- Compile to assembly
- Link with runtime
- Load on system
- Run and test

**Hardware integration**:
- Use runtime routines
- Call hardware functions
- Access I/O devices
- Use graphics/sound
- Real-time interaction
