# Year 5, Term 1: Assessment Rubric

## Overview

This rubric assesses student progress through Term 1 of Year 5, covering PCB design, assembly, and system integration.

## Assessment Components

### 1. PCB Design (Weeks 1-5) - 35%

#### Practical Work
- **Schematic Finalization** (12%)
  - Finalizes all schematics
  - Designs power distribution
  - Adds decoupling
  - Documents design
  - **Exceeds**: Optimized design

- **PCB Layout** (12%)
  - Creates PCB layout
  - Places components properly
  - Routes traces correctly
  - Adds ground planes
  - **Exceeds**: Advanced techniques

- **Gerber Generation** (11%)
  - Generates Gerber files
  - Verifies all layers
  - Passes design review
  - Ready for ordering
  - **Exceeds**: Publication-quality

#### Understanding
- Explains PCB design clearly
- Understands power distribution
- Understands signal integrity
- Can create professional layouts

### 2. PCB Assembly (Weeks 6-9) - 35%

#### Practical Work
- **Component Assembly** (12%)
  - Assembles components correctly
  - Uses ESD protection
  - Good soldering quality
  - Documents assembly
  - **Exceeds**: Professional quality

- **Continuity and Smoke Tests** (12%)
  - Performs continuity tests
  - Passes smoke tests
  - Verifies voltages
  - Documents results
  - **Exceeds**: Comprehensive testing

- **Subsystem Testing** (11%)
  - Tests all subsystems
  - Documents test results
  - Fixes issues
  - Verifies operation
  - **Exceeds**: Advanced testing

#### Understanding
- Explains assembly procedures
- Understands ESD protection
- Understands testing procedures
- Can debug issues

### 3. System Integration (Weeks 10-12) - 30%

#### Practical Work
- **Board Integration** (10%)
  - Integrates all boards
  - Verifies connections
  - Secures mounting
  - Documents integration
  - **Exceeds**: Optimized integration

- **System Testing** (10%)
  - Tests complete system
  - Verifies all functionality
  - Tests video/audio
  - Tests SD card
  - **Exceeds**: Comprehensive testing

- **Acceptance Testing** (10%)
  - Passes acceptance tests
  - Documents results
  - System verified
  - Ready for software
  - **Exceeds**: Performance optimized

#### Understanding
- Explains integration process
- Understands testing procedures
- Can debug system issues
- Understands system operation

## Overall Assessment

### Grade Descriptors

**Excellent (A)**:
- All practical work completed successfully
- Deep understanding of PCB design
- Excellent documentation
- Complete system working
- Helps other students

**Good (B)**:
- Most practical work completed successfully
- Good understanding of PCB design
- Good documentation
- System mostly working
- Can apply concepts with guidance

**Satisfactory (C)**:
- Basic practical work completed
- Basic understanding of PCB design
- Basic documentation
- System partially working
- Needs guidance to apply concepts

**Needs Improvement (D)**:
- Some practical work incomplete
- Limited understanding
- Incomplete documentation
- System not working
- Significant guidance needed

## Assessment Methods

### Formative Assessment
- Weekly design reviews
- Assembly checkpoints
- Integration testing
- Peer reviews

### Summative Assessment
- PCB design project
- Assembly and testing project
- System integration project
- Complete system demonstration

## Documentation Requirements

### Lab Reports
- Design decisions
- Assembly logs
- Test procedures
- Test results
- Issues and fixes
- System verification

## Extension Opportunities

Students can extend their work by:
- Advanced layout techniques
- Signal integrity optimization
- Performance optimization
- Help other students
