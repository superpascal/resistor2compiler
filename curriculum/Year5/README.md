# Year 5: System Integration and Expansion (Age 17)

## Overview

Year 5 transitions to the 65C816 16-bit microprocessor system. Students design and build a complete 16-bit computer with expanded capabilities.

## Learning Outcomes

By the end of Year 5, students will:
- Design a 65C816-based system
- Build 16-bit computer on breadboards
- Program in 65C816 assembly
- Integrate I/O devices
- Develop monitor program
- Understand memory banking

## Year Structure

- **Term 1**: PCB Design and Assembly (Weeks 1-12)
  - Weeks 1-5: Finalizing PCB Schematics and Layout
  - Weeks 6-9: PCB Assembly and Unit Testing
  - Weeks 10-12: System Integration on PCB
- **Term 2**: High-Level Language & OS (Weeks 13-24)
  - Weeks 13-16: SuperPascal Compiler Implementation
  - Weeks 17-20: Pascal Program Development & Demos
  - Weeks 21-24: Operating System/Monitor Extensions

## Key Topics

### Term 1: Hardware
- 65C816 system design
- Memory and banking
- I/O subsystems (VIA, ACIA)
- Graphics and sound research
- Breadboard prototype

### Term 2: Software
- Monitor program development
- Assembler and toolchain
- Pascal runtime preparation
- System integration

## Prerequisites

- Completed Year 4
- Understanding of CPU architecture
- Basic assembly language concepts

## Assessment

- System design review
- Working 16-bit prototype
- Monitor program
- Assembly programs
- Documentation
