# Year 5, Term 3: Advanced Software and A-Level Preparation

## Overview

Term 3 of Year 5 focuses on advanced software development, PascalOS development, and A-Level project preparation. Students extend their system capabilities and prepare for A-Level assessments.

## Learning Outcomes

By the end of Term 3, students will:
- Develop PascalOS operating system features
- Create advanced Pascal programs
- Implement system services and APIs
- Prepare A-Level project documentation
- Demonstrate complete system capabilities

## Week Structure

### Weeks 25-28: PascalOS Development
- Extend monitor to PascalOS
- Implement file system support
- Add program management
- Implement basic multitasking (cooperative)
- Create system services API

### Weeks 29-32: Advanced Pascal Programming
- Advanced Pascal features
- Graphics programming
- Sound programming
- Game development
- Application development

### Weeks 33-36: A-Level Project Development
- Complete A-Level project
- System documentation
- Project presentation preparation
- Portfolio completion
- Final system demonstration

## Materials

### Components
- Complete 65C816 breadboard system
- Serial interface
- Development PC

### Tools
- SuperPascal compiler
- Assembler and toolchain
- Text editor
- Documentation tools
- Presentation software

## Assessment

- **Formative**: Weekly development checkpoints
- **Summative**: PascalOS features (Weeks 25-28)
- **Summative**: Advanced programs (Weeks 29-32)
- **Final**: A-Level project and presentation (Weeks 33-36)

## Prerequisites

Students must have completed Term 2 and have:
- Working SuperPascal compiler
- Pascal programs running
- Understanding of system architecture
- Experience with Pascal programming

## Key Concepts

- **Operating System**: System services and APIs
- **Advanced Programming**: Complex applications
- **Project Management**: A-Level project development
- **Documentation**: Professional system documentation
- **Presentation**: Demonstrating system capabilities

## Milestones

- **Week 28**: PascalOS features implemented
- **Week 32**: Advanced programs complete
- **Week 36**: A-Level project complete and presented
